现代 Android 开发需要掌握哪些技能

发表于 2025-06-25 更新于 2025-06-28 分类于 Android

现代 Android 开发已经不仅仅是写 Java 或 Kotlin 代码那么简单，它要求开发者具备跨层能力，掌握现代架构思想、异步编程模式、性能优化技巧以及与平台生态相关的工具链。以下是一份系统化的技能清单，分层说明现代 Android 开发需要掌握的内容：

✅ 一、语言基础

  • Kotlin（推荐）

    • 协程（Coroutines）、Flow
    • 扩展函数、委托属性、密封类、泛型
    • Kotlin DSL（Gradle、Jetpack Compose）

  • Java（用于阅读老项目或与 SDK 接口打交道）

✅ 二、UI 开发

1. 现代 UI：Jetpack Compose

  • Composable 函数、状态管理（State/MutableState/StateFlow）
  • SideEffect APIs：LaunchedEffect、rememberCoroutineScope 等
  • 动画、主题、自定义组件
  • Navigation-Compose 路由导航

2. 传统 UI：View System

  • ConstraintLayout、RecyclerView、ViewPager2
  • 自定义 View、动画系统
  • XML 布局 + DataBinding/ViewBinding

✅ 三、架构模式

  • MVVM（现代主流）
  • MVI（Compose 结合使用多）
  • Jetpack 官方组件：ViewModel、LiveData、StateFlow
  • 单一数据源（Single source of truth）
  • Repository 模式
  • 使用 Hilt/Dagger 进行依赖注入

✅ 四、异步编程

  • Kotlin 协程：suspend 函数、launch、async
  • Flow 与 StateFlow：背压、冷流、热流
  • 生命周期感知的协程：lifecycleScope、repeatOnLifecycle

✅ 五、Jetpack 组件生态

  • Room：数据库操作（替代 SQLite）
  • WorkManager：后台任务调度
  • DataStore：替代 SharedPreferences 的新方案
  • Paging3：大数据分页加载
  • Navigation：Fragment/Compose 路由管理

✅ 六、网络和数据

  • Retrofit + OkHttp（网络请求）
  • Moshi / Gson（JSON 序列化）
  • Ktor Client（现代异步 HTTP 客户端，尤其适用于 Kotlin Multiplatform）

✅ 七、依赖注入（DI）

  • Hilt（官方推荐）
  • Dagger2（老项目仍大量使用）
  • Koin（Kotlin 优雅方案，Compose 项目中较多）

✅ 八、测试能力

  • 单元测试：Junit、MockK/Mockito
  • UI 测试：Espresso、UI Automator
  • Compose UI 测试：composeTestRule
  • Instrumentation 测试：测试真实设备行为

✅ 九、性能优化

  • 启动优化、内存优化（LeakCanary、Profiler）
  • 帧率卡顿（Choreographer/FrameMetrics）
  • ANR 检测
  • 多线程优化、协程调度优化

✅ 十、项目构建与管理

  • Gradle 脚本（Groovy / Kotlin DSL）
  • 多模块化项目（Modularization）
  • 构建速度优化、依赖版本管理（Version Catalogs）
  • CI/CD：GitHub Actions、GitLab CI、Bitrise、Firebase App Distribution

✅ 十一、平台能力

  • 权限系统（动态权限申请）
  • 相机、定位、蓝牙、音视频处理（如 Media3、ExoPlayer）
  • 电池优化、前后台切换、通知
  • 兼容性适配（折叠屏、多窗口、大屏幕）

✅ 十二、跨平台与拓展方向

  • Kotlin Multiplatform Mobile (KMM)
  • Flutter、React Native（若团队使用）
  • Compose Multiplatform（支持桌面和 Web）

✅ 十三、发布与应用管理

  • Google Play Console 上架流程
  • 应用签名、Bundle（AAB）打包
  • ProGuard / R8 混淆与优化
  • 崩溃分析：Firebase Crashlytics、Bugsnag 等

✅ 十四、开发工具

  • Android Studio、ADB、Profiler
  • Layout Inspector、Network Inspector
  • AS 插件如：Lint、LeakCanary、StrictMode
  • 命令行构建与调试（Gradle CLI、adb shell）

如果你是初学者，建议优先掌握：

  1. Kotlin 语言（熟练使用）
  2. Jetpack Compose（推荐）
  3. MVVM 架构 + ViewModel + Flow
  4. 网络请求（Retrofit）+ 本地缓存（Room）
  5. 简单项目模块化
  6. 日常调试和性能工具（如 Profiler）

如果你是进阶开发者，建议往 性能、架构、跨平台、CI/CD 方向深入。
