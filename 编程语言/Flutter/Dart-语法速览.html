1.1 程序结构

2.1 面向对象编程 (OOP)

继承

接口和抽象类

Dart 没有显式的接口声明，类可以通过 implements 来实现接口。

Dart 提供了空安全功能，防止 null 相关的错误。

2.5 泛型

Dart 支持泛型，用于提高代码复用性和类型安全性。

泛型列表

泛型类

2.6 扩展方法

扩展方法允许为现有类添加新功能。

2.7 Mixin

Mixin 是一种复用代码的方式，常用于类的功能扩展。

2.8 元编程 (Reflection)

Dart 提供 dart:mirrors 来支持元编程。

2.9 Isolate 并发

Dart 的 Isolate 提供了独立的内存空间和线程模型。

总结

基础语法

  1. Dart 支持强类型变量、空安全机制和简洁的控制语句。
  2. 提供灵活的函数声明，包括匿名函数、命名参数和默认参数。

高级特性

  1. OOP: Dart 支持类、继承、抽象类和接口。
  2. 异步编程: 使用 Future 和 Stream 处理异步任务。
  3. 泛型: 提高代码复用性和类型安全性。
  4. 扩展与 Mixin: 为类添加扩展功能，增强代码复用。
