1. HTML 的基本结构

HTML 是超文本标记语言，用于定义网页的内容。

基本文档结构：

元素组成：

  • 标签：如 <h1> 和 </h1>。
  • 属性：如 lang="en"。
  • 内容：如 这是一个标题。

段落和换行

  • 段落：<p>
  • 换行：<br>

文本格式化

3. 超链接

超链接使用 <a> 标签。

  • href：链接地址。
  • target="_blank"：新窗口打开。

4. 图像

图像使用 <img> 标签。

  • src：图片路径。
  • alt：图片描述（用于无障碍和加载失败时显示）。
  • width 和 height：宽度和高度。

无序列表

有序列表

6. 表格

表格使用 <table> 标签。

  • <th>：表头。
  • <tr>：表行。
  • <td>：表格单元格。

7. 表单

表单使用 <form> 标签，用于收集用户输入。

  • action：提交地址。
  • method：提交方式（get 或 post）。
  • 常见输入类型：text、password、email、number、checkbox、radio。

9. HTML5 新特性

语义化标签

  • **<header>**：定义页面头部。
  • **<footer>**：定义页面尾部。
  • **<article>**：独立的文章。
  • **<section>**：页面的独立部分。
  • **<nav>**：导航菜单。
  • **<aside>**：侧边栏。

本地存储（Local Storage 和 Session Storage）

11. 特殊字符

HTML 支持的常用特殊字符：

  • &lt;：小于号 <
  • &gt;：大于号 >
  • &amp;：和号 &
  • &nbsp;：空格
  • &copy;：版权符号 ©
  • &reg;：注册商标符号 ®

12. 注释

这些 HTML 基本语法涵盖了常用的标签和属性，为进一步学习 CSS 和 JavaScript 提供了基础。
