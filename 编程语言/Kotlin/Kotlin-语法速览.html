Kotlin 语法速览

发表于 2025-05-27 更新于 2025-06-03 分类于 编程语言 ， Kotlin

1. 变量和常量

  • val：不可变变量（只读，类似 final）
  • var：可变变量
  • 类型推断：Kotlin 支持自动推断变量类型

2. 基本数据类型

  • 常见类型：Int, Double, String, Boolean, List, Map 等
  • 无基本类型与引用类型的区分，全部是对象

3. 控制流：if 表达式

  • if 在 Kotlin 中是表达式，可以返回值
  • 替代三元运算符（Kotlin 无三元运算符）

4. 控制流：when 表达式

  • 类似 switch，但更强大，支持多种条件
  • 也是表达式，可返回值

5. 循环

  • for：用于迭代集合或范围
  • while 和 do-while：与 Java 类似

6. 函数

  • 使用 fun 关键字
  • 支持默认参数、命名参数、单表达式函数

7. 类和对象

  • 使用 class 定义类
  • 属性自动生成 getter/setter
  • 支持构造函数、主构造函数、初始化块

8. 继承和接口

  • 默认类是 final，需用 open 允许继承
  • 接口使用 interface，实现用 :

9. 空安全

  • 可空类型用 ? 表示
  • 安全调用 ?.，Elvis 运算符 ?:，非空断言 !!

10. 集合

  • 支持 List, Set, Map
  • 提供不可变和可变集合：listOf, mutableListOf 等

11. Lambda 表达式和匿名函数

  • Lambda 语法：{ 参数 -> 表达式 }
  • 常用于集合操作

12. 扩展函数

  • 为现有类添加方法，无需继承

13. 数据类

  • 使用 data class 自动生成 toString, equals, hashCode 等

14. 协程

  • 用于异步编程，需引入 kotlinx.coroutines
  • 使用 launch 或 async 启动协程

注意：协程示例需要添加 kotlinx-coroutines-core 依赖到项目中，否则无法运行。
