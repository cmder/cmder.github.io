将公钥复制到 GitHub

将 SSH 公钥（id_rsa.pub）添加到您的 Git 帐户

  • 打开 GitHub 网站并登录到您的帐户。
  • 单击页面右上角的头像，然后单击 “Settings”。
  • 在左侧导航栏中，单击 “SSH and GPG keys”。
  • 单击 “New SSH key”。
  • 将复制的 SSH 公钥粘贴到 “Key” 文本框中，并为 SSH 密钥添加一个标题。
  • 单击 “Add SSH key”。
